## Russian Bank (Crapette / Zank-Patience)

A browser-playable, two-player hot-seat card game in [`russian-bank/`](russian-bank/), modeled on the traditional rules (separate decks, Reserve/Hand/Waste piles, shared Houses and Foundations, loading the opponent's Reserve/Waste). Open `russian-bank/index.html` in a browser — no build step required — or use `russian-bank/russian-bank-standalone.html` for a single self-contained file (handy for mobile/offline). See the in-app "Rules" button for full details.

This implementation skips the traditional "Stop!" foul-calling mechanic and the compulsory-move priority order — players are free to make any legal move in any order.
